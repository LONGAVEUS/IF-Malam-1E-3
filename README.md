## Cara Install Website LocalHostnya?

1. Download source code lewat zip
2. Ekstract ke file perangkat lunak Localhost anda seperti (www. atau htdocs)
3. Untuk melakukan pengecekkan, bisa menggunakan text editor milik anda (NotePad, VsCode, dll)
4. Jalankan terlebih dahulu perangakt lunak localhoast anda.
5. • 	Jalankan XAMPP/Laragon → aktifkan MySQL.
   • 	Buka browser → akses **http://localhost/phpmyadmin**
   • 	Sesudah dihalaman, click Import.
   •  Klik tab Import di menu bagian atas.
   •  Klik tombol Choose File dan pilih file .sql dari komputer Anda.
   •  Pastikan format yang terpilih adalah SQL. Bisa didownload sqlnya (db_notulen.sql) di repository/file zip sebelumnya.
   •  Scroll ke bawah dan klik tombol Import atau Go.
6. Gunakan URL access ini untuk dapat mengakses localhost website http://localhost/IF-MALAM-1E-3/
7. Anda berhasil menjalankan Aplikasi

## Cara penggunaan aplikasi pencatatan notulen?
 Penggunaan Notulen untuk berbagai Role:
 Tamu --> Menerima notulen --> Isi kehadiran ---> **RAPAT SELESAI** --> Download notulen
 Notulis --> Mmebuat notulen --> Mengirim Kehadiran kepeserta --> **Menerima Kehadiran** --> **Rapat selesai** -->   Notulen dapat di finalisasi --> Download Notulen.
 Admin --> Dapat melihat perkembangan Rapat
       --> Management peserta

Jadi dalam penggunaan aplikasi terdapat perbedaan role yang menentukan jalannya notulen.
